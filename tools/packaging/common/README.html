Pixel Wheels

Welcome to Pixel Wheels! Pixel Wheels is a retro top-down race game. You can play alone or with a friend.

It works on Linux, Intel-based macOS, Windows and Android.

üì¶ Installation

You can start the game directly from this folder, but if you want to install it, here is how.

Linux

Run the install.sh script to install it on your system. This will make the game available from your desktop menu.

macOS

Drag the "Pixel Wheels" bundle to your "Application" folder.

Windows

There is no automated installation procedure for now, but you can copy this folder in the folder of your choice and create a shortcut to the pixelwheels.exe executable.

üèé Controls

The user interface and the vehicle can be controlled using either gamepads or keyboard.

Vehicles accelerate automatically.

‚å® Keyboard

To navigate the menus and control the first player vehicle, use the arrow keys and the Space key to select menus or activate bonuses.

In 2-player mode, the keys for the second player are:

  • X: Left
  • V: Right
  • D: Up
  • C: Down / brake
  • Left control: activate bonuses

Keys can be configured from the Settings menu.

üïπ Gamepads

You can configure mapping of the gamepad buttons in the configuration menu.
